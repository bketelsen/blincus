# blincus shell

Open a shell in an instance

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | s

## Usage

```bash
blincus shell NAME [OPTIONS]
```

## Examples

```bash
blincus shell mydevctr --login
```

```bash
blincus shell mydevctr --root --no-login
```

```bash
blincus shell mydevctr
```

## Arguments

#### *NAME*

Instance name
